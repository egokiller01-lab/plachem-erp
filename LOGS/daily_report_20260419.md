## 1. 주요 작업 성과

### ✅ 시스템 봉쇄 해제 및 권한 복구
- **프로필 RLS 정책 수정**: `profiles` 테이블의 무한 재귀 오류를 해결했습니다.
- **편집 권한 로직 개선**: 신규 전표 작성 시 본인 여부 검증을 생략(canEdit)하여, 모든 모듈에서 새 문서 작성이 가능해졌습니다.

### ✅ 실무 중심 컴팩트 UI 개편
- **정보 밀도 극대화**: 상하좌우 여백을 약 50% 축소(globals.css)하여 한 화면에 더 많은 데이터가 보이도록 조정했습니다.
- **상단 헤더 압축**: 전표 상단(Basic Info) 공간을 최소화하여 데이터 그리드 가시 영역을 확보했습니다.

### ✅ 제품 선택 로직 재구축 (안정성 확보)
- **ID 기반 바인딩 기술**: 모든 매칭 기준을 `product_id`로 일원화하여 저장 정확도를 높였습니다.
- **Selector/Display 분리**: 입력창(검색)과 정보 표시(3줄 정적 텍스트)를 분리하여 테이블 레이아웃을 안정화했습니다.
- **로딩 순서 강제**: 마스터 데이터를 먼저 다 부른 후 전표를 세팅하도록 수정하여 품명 누락을 차단했습니다.
